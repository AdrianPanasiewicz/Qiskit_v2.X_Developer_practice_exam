# IBM Certified Quantum Computation using Qiskit v2.X Developer Associate — Practice Exam

This repository contains a practice exam notebook for the IBM Certified Quantum Computation using Qiskit v2.X Developer - Associate certification.

The notebook includes:
- practice multiple-choice questions,
- answers,
- short explanations for selected questions,
- coverage of the main exam domains such as quantum operations, circuit visualization, circuit creation, circuit execution, Sampler, Estimator, results analysis, and OpenQASM.

It is intended as a revision resource for the official certification exam, which contains 68 questions and must be completed in 90 minutes.

Important: please view this notebook in **VS Code** or **PyCharm**, not in the default GitHub notebook preview, because it uses HTML to hide and reveal answers.
